# Whack-a-Mole iOS

Whack-a-Mole is a SpriteKit tap game with a title screen, randomized mole placement, scoring, timer pressure, and saved high scores.

## Features

- SpriteKit title and game scenes
- Random mole placement across multiple holes
- Touch handling for hits and misses
- Score, high score, and countdown timer
- Reset flow
- Saved-game persistence
- Custom game artwork

## Tech Stack

- Swift
- SpriteKit
- SKScene
- SKSpriteNode
- SKLabelNode
- SKAction
- NSKeyedArchiver persistence

## Run Locally

1. Open `WhackAMole.xcodeproj` in Xcode.
2. Run the app in an iOS Simulator.
